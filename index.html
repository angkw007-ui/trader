⚙️ Google Sheets 연동 설치 가이드
1
Google Sheets 만들기
sheets.new 접속 → 새 시트 생성
URL에서 Spreadsheet ID 복사:
https://docs.google.com/spreadsheets/d/[여기가 ID]/edit
2
Apps Script 열기
Google Sheets 메뉴 → 확장 프로그램 → Apps Script
3
Code.gs 코드 붙여넣기
기존 내용 전체 삭제 → Code.gs 파일 내용 붙여넣기
SHEET_ID = '복사한 스프레드시트 ID' 로 수정 후 저장(Ctrl+S)
4
웹앱으로 배포
오른쪽 상단 배포 → 새 배포 → 유형: 웹 앱
· 설명: 매매일지
· 실행 계정: 나
· 액세스 권한: 모든 사용자 (Anyone)
→ 배포 클릭 → 권한 허용 → 웹 앱 URL 복사
5
URL 입력하고 연결
상단 노란 배너에 복사한 URL 붙여넣기 → 저장
연결 성공 시 상단에 🟢 초록 점 표시
6
GitHub Pages 업로드 (어디서든 접속)
github.com → New repo → index.html로 업로드
Settings → Pages → main → Save
→ https://아이디.github.io/repo이름/ 으로 접속
✓ 이제 PC·스마트폰 어디서든 같은 구글 시트에 데이터가 저장됩니다!
